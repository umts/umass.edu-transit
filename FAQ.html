The Bus Tracker


What is the Bus Tracker?

The Bus Tracker is a web based application that shows buses and bus stops for UMass Transit. This application is a CmpSci experiment and as a result, not all buses will always show current information. To read more about it, click here

How do I use the Bus Tracker?

The Bus Tracker is a google map which shows UMass Transit buses and bus stops. When using The Bus Tracker, click the icons to get more information.

Current Location
Scheduled Location (displayed when communication is lost for 2 minutes)
A bus stop

What is the difference between "Current Location" and "Scheduled Location"?

Because the Bus Tracker is an experiment, the equipment is not always operational. Also, the equipment does not always have 100% reception (much like a cell phone). If the icon is green, it is showing the current location. If the icon is grey, it is showing the scheduled location.


What if I have a problem using the Bus Tracker?

Please contact us here.


Using Google Transit with the PVTA


What is Google Transit?

Google Transit is a mapping program that allows you to enter in two location, and get directions between those two points.

What type of directions does Google Transit offer?

Google Transit allows the user to map a trip using the following types of transportation:

Pedestrian
Bicycle
Bus
